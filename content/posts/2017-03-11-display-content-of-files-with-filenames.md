Use the following linux command to display the contents of several files with their filenames prefixed.

`tail -n +1 file1.txt file2.txt file3.txt`

Displays as

`<span class="s1">==> file1.txt <==</span>`

File 1 contents

==> file2.txt <==

File 3 contents

==> file3.txt <==

File 3 contents
